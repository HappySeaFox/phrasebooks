Phrasebook
----------

Author: Dmitry Baryshev <linuxsquirrel.dev@gmail.com>
License: GNU GPLv3+ (see LICENSE.txt)

The purpose of this applications is to manage phrasebooks
and load the specified word or text into an external dictionary.

The application is intended to a person who'd like to learn a foreign
vocabulary but his/her favorite dictionary doesn't support advanced
phrasebooks (like ABBYY Lingvo).

If your favorite dictionary supports advanced phrasebooks, then
you don't need this application.
